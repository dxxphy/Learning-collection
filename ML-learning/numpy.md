## 1. 核心结构：ndarray (N-dimensional Array)

NumPy 的核心是 N 维数组对象。

### 1.1 维度识别法（剥洋葱法则）

**核心技巧**：看最外层有几层中括号 `[`，就是几维数组。

| 维度 | 形状示例 `shape` | 代码外观 | 几何含义 |
| --- | --- | --- | --- |
| **一维** | `(3,)` | `[1, 2, 3]` | **清单/线**。无行列之分，只有长度。 |
| **二维** | `(2, 3)` | `[[1, 2, 3], [4, 5, 6]]` | **表格/面**。有行 (axis 0) 和 列 (axis 1)。 |
| **三维** | `(2, 3, 4)` | `[[[...], [...]]]` | **书本/体**。页 (axis 0) - 行 (axis 1) - 列 (axis 2)。 |

### 1.2 重要属性

```python
a = np.array([[1, 2, 3], [4, 5, 6]])
print(a.ndim)   # 2 (维度数量)
print(a.shape)  # (2, 3) (形状：2行3列)
print(a.size)   # 6 (元素总数)
print(a.dtype)  # dtype('int64') (数据类型)

```

## 2. 数组创建 (Creation)

### 2.1 常用创建函数

| 函数 | 参数含义 | 示例 | 说明 |
| --- | --- | --- | --- |
| `np.array()` | `object`: 列表或元组 | `np.array([1, 2])` | 从 Python 列表转换 |
| `np.zeros()` | `shape`: **元组** (行, 列) | `np.zeros((3, 4))` | 全 0 数组，用于占位 |
| `np.ones()` | `shape`: **元组** | `np.ones((2, 3))` | 全 1 数组 |
| `np.arange()` | `start, stop, step`: **数字** | `np.arange(0, 10, 2)` | 类似 range，左闭右开 `[0, 2, 4, 6, 8]` |
| `np.linspace()` | `start, stop, num`: **数字** | `np.linspace(0, 1, 5)` | 在 0 到 1 之间**均匀**生成 5 个数 |
| `np.eye()` | `N`: 行数/列数 | `np.eye(3)` | 创建单位矩阵 (对角线为1) |

### 2.2 高级创建：`fromfunction`

根据坐标索引生成数值。

```python
# 参数 function: 接收坐标 (x, y)，返回该位置的值
# 参数 shape: (5, 4) 决定数组大小
def f(x, y): return 10 * x + y
b = np.fromfunction(f, (5, 4), dtype=int)

```

> **注意**：
> * 描述**形状**时（如 `zeros`, `reshape`），通常传**元组** `(row, col)`。
> * 描述**范围/数量**时（如 `arange`, `linspace`），通常直接传**数字**参数。

## 3. 广播机制 (Broadcasting) 

**核心规则**：NumPy 自动处理不同形状数组的运算。
**判断流程**：**右对齐 -> 检查兼容 -> 自动拉伸**

1. **右对齐**：将两个数组的 shape 向右靠齐。
2. **检查**：从右向左逐个维度对比，满足以下任一条件即兼容：
* 两个维度数字**相等**。
* 其中一个维度是 **1**。


3. **拉伸**：如果不相等且其中一个是 1，则将 **1** 拉伸（复制）成另一个数字的大小。

### 经典案例解析：`(3, 2)` 与 `(2,)`

```python
A = np.ones((3, 2))  # Shape: (3, 2)
B = np.array([1, 1]) # Shape: (2,)

# 运算 A + B 过程：
# 1. B 补全维度 -> (1, 2)
# 2. B 拉伸维度 -> (3, 2) (把 [1,1] 复制了 3 行)
# 3. 相加

```

> **陷阱**：如果形状是 `(3, 2)` 和 `(3,)`，右对齐后是 `2` 和 `3` 对比，不相等且无 1，**报错**。

## 4. 索引与切片 (Indexing & Slicing)

### 4.1 基础切片

符号 `:` 代表“全部”或“范围”。

```python
a[1:5]    # 取索引 1 到 4
a[::-1]   # 倒序
a[1, :]   # 取第 1 行，所有列
a[:, 2]   # 取所有行，第 2 列

```

### 4.2 布尔索引 (Boolean Indexing)

用于筛选、过滤数据（非常常用）。

```python
a = np.array([10, 20, 30, 40])
mask = a > 25          # [False, False, True, True]
print(a[mask])         # [30, 40]
a[a < 20] = 0          # 将小于 20 的数直接修改为 0

```

## 6. 统计与计算方法

绝大多数聚合函数都有 `axis` 参数。

### 6.1 Axis (轴) 的含义

* **axis=0**：沿着第 0 轴操作（垂直方向/跨行）。
* **axis=1**：沿着第 1 轴操作（水平方向/跨列）。

### 6.2 常用函数

| 函数 | 作用 | 示例 |
| --- | --- | --- |
| `sum()` | 求和 | `a.sum(axis=0)` (列求和) |
| `mean()` | 平均值 | `a.mean()` |
| `std()` | 标准差 | `a.std(axis=1)` (行标准差) | |
| `var()` | 方差 | `a.var()` |
| `max() / min()` | 最值 | `a.max()` |
| `argmax() / argmin()` | **返回最值的索引** | `a.argmax(axis=1)` (找每一行最大值的位置) |
| `cumsum()` | 累积和 | `a.cumsum()` (斐波那契式的累加) |
| `argsort()` | 返回排序后的索引 | `idx = a.argsort()` (常用于间接排序) |

## 7. 随机数 (`np.random`) 与 文件存取

### 7.1 随机数

使用前建议设置种子：`np.random.seed(42)`

| 函数 | 参数 | 形状写法 | 说明 |
| --- | --- | --- | --- |
| `rand()` | `d0, d1...` | `rand(3, 2)` | [0, 1) 均匀分布 |
| `randn()` | `d0, d1...` | `randn(3, 2)` | 标准正态分布 (均值0，方差1) |
| `randint()` | `low, high, size` | `size=(3, 2)` | 生成随机整数 |
| `choice()` | `a, size` | `size=5` | 从数组 `a` 中随机抽取 |
| `shuffle()` | `x` | 直接传入数组 | **原地打乱**顺序 |

### 7.2 文件存取

* **推荐**：`np.save('file.npy', arr)` / `np.load('file.npy')` (保留形状和类型，速度快)。
* **文本**：`np.savetxt('file.csv', arr)` / `np.loadtxt('file.csv')` (通用性强，但丢失形状)。

## 8. 实战代码模板：欧几里得距离

综合运用了创建、广播、数学运算、聚合。

```python
# 计算 points (3个点) 到 target (1个点) 的距离
points = np.array([[0, 0], [3, 4], [1, 1]]) # (3, 2)
target = np.array([1, 1])                   # (2,)

# 一行代码实现：
# 1. (points - target): 广播相减
# 2. **2: 平方
# 3. sum(axis=1): 每一行的坐标差平方求和
# 4. sqrt: 开根号
dist = np.sqrt(np.sum((points - target)**2, axis=1))

```

## 9. 避坑指南 (易错点总结)

1. **一维数组**：`shape` 是 `(n,)`，它既不是行向量也不是列向量。如果需要矩阵乘法，建议 `reshape(1, n)` 或 `reshape(n, 1)`。
2. **参数括号**：
* `np.zeros((2, 3))`：需要两层括号（外层是函数调用，内层是元组）。
* `np.arange(0, 10)`：不需要内层括号（参数独立）。


3. **矩阵乘法**：使用 `@` 或 `.dot()`，千万不要用 `*`（那是对应元素相乘）。
4. **Copy vs View**：切片返回的是**视图**，修改切片会改动原数组。如果不希望这样，请使用 `.copy()`。
